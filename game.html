Game

I followed a Javascript tutorial from W3 Schools to put this game on my website. I enhanced it by creating my own butterfly sprite, slightly increasing the gravity, and adding new colors to the game's assets with CSS.

Press FLY repeatedly to make the butterfly fly. Avoid the green vines!

The works on this student portfolio have been necessarily resized and cropped from their original versions in order to fit the website dimensions.
